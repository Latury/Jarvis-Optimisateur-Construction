**Date de sortie** : 13 novembre 2025
**Statut** : Phase de développement initial

### 🎯 Objectif de cette version

Mise en place de toute l'infrastructure du projet et préparation des bases pour le développement des fonctionnalités principales.

---

### ✨ Nouveautés

#### 📁 Structure du projet
- **Création de l'architecture complète** du projet avec une organisation claire et professionnelle
- **Dossiers organisés** :
  - `css/` pour tous les fichiers de styles
  - `js/` pour tout le code JavaScript
  - `ressources/icones/` pour toutes les images du jeu

#### 🎨 Ressources graphiques
- **Import des icônes** depuis l'ancien projet :
  - 🔧 Icônes des **Cogs** (engrenages) - toutes les variantes
  - 🎩 Icônes des **chapeaux** pour les personnages
  - 👤 Icône de la **tête** du personnage
  - 📦 Fichiers de référence archivés pour usage futur

#### 📚 Documentation
- **README professionnel** avec :
  - Description complète du projet
  - Guide d'installation et d'utilisation
  - Roadmap détaillée des prochaines versions
  - Badges de statut et liens utiles
- **CHANGELOG technique** pour suivre toutes les modifications
- **PATCHNOTES** pour les utilisateurs (ce fichier !)

#### ⚙️ Configuration
- **Dépôt GitHub** configuré et opérationnel
- **Système de versioning** professionnel (Semantic Versioning)
- Préparation pour le **déploiement sur GitHub Pages**

### 🔨 En cours de développement

Les fonctionnalités suivantes sont en préparation pour les prochaines versions :

#### Version 0.2.0 (Prochaine)
- 🖥️ **Interface utilisateur complète**
  - Tableau de construction interactif (12×8 cases)
  - Inventaire des Cogs avec pagination
  - Onglets de navigation (Menu, Options, Sauvegarde, Aide, Logs)

  - Algorithme génétique avancé

  - Comparaison avant/après
