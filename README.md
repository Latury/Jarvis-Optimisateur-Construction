## 📋 Description

**Jarvis - Optimisateur de Construction** est un outil web permettant d'optimiser le placement des **Cogs** (engrenages) dans le système de construction du jeu **Legend of Idleon** (Monde 3).

Cet optimiseur utilise des algorithmes génétiques avancés pour calculer la meilleure disposition des Cogs sur votre plateau de construction afin de maximiser vos bonus (taux de construction, XP, vitesse de flags).
