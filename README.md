## 📋 Description
Jarvis est un outil web pour optimiser le placement des Cogs dans Legend of Idleon (Monde 3).
Il utilise un algorithme génétique pour maximiser taux de construction, XP, et vitesse de flags.

---

## ✨ Fonctionnalités - v0.1.0 (Développement initial)
- Structure projet et ressources graphiques importées et organisées
- Interface web basique avec gestion des modals et logs console
- Chargement automatique du modal « Chargeur de données »
- Pagination d’inventaire et système d’étapes dynamiques
- Log console pour suivre les actions et erreurs

cd Jarvis-Optimisateur-Construction

Ouvrir index.html dans un navigateur moderne
